FIRST STEP SCHOOL — WEBSITE
===========================
Teshie Nungua Estates, Accra, Ghana

A complete, professional static website built with pure HTML & CSS.
No build tools, no dependencies — just open the files in any browser.

PAGES (all linked in the header & footer)
----------------------------------------
  index.html      — Home
  about.html      — About / Mission / Vision
  academics.html  — Programs & curriculum
  admissions.html — How to apply
  gallery.html    — Photo gallery
  contact.html    — Contact form + map

STRUCTURE
---------
  *.html          — Pages (root)
  css/style.css   — All styles (sea blue & white theme)
  images/         — Your photos go here (see images/README.txt)

HOW TO USE
----------
1. Open index.html in your web browser to preview the site.
2. Add your own photos to the images/ folder using the filenames listed
   in images/README.txt.
3. To publish online, upload the whole folder to any web host
   (Netlify, Vercel, GitHub Pages, your cPanel, etc.).

COLOURS
-------
Sea blue (#0e7c9b) and white — editable via the CSS variables at the
top of css/style.css (the --sea variables).
